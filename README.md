# alias-tool

## Features

1. scss: `@import "path";`  `@use "path";`  `url(...) ` alias
2. js / ts: `import "path"` asset alias and fix path with search

## Extension Settings

`alias-tool.mappings`:

```
{
    "alias-tool.mappings": {
        "app": "frontend/app",
        "@client": "frontend/app",
        "~@client": "frontend/app"
    }
}
```
